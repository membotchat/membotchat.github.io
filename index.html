Installation Instructions

Follow these simple steps to install BetterCountdown on your Mac:

Step 1: Download & Install

  1. Download the BetterCountdown.dmg file
  2. Double-click the downloaded DMG file to mount it
  3. Drag BetterCountdown.app to your Applications folder
  4. Eject the DMG file

⚠️ IMPORTANT - First Launch Only:

Because this app is not notarized by Apple (which costs $99/year), macOS will block it the first time. This is normal! Here's how to open it:

Step 2: Bypass Gatekeeper (First Time Only)

  1. Try to open the app normally (you'll see a warning)
  2. Go to System Settings → Privacy & Security
  3. Scroll down and click "Open Anyway" next to BetterCountdown
  4. Click "Open" to confirm

✅ This App is Safe!

Apple requires a $99/year developer account to "notarize" apps. This free app hasn't been notarized, so macOS shows a precautionary warning. The app is completely safe to use, and you'll only see this warning once. After the first launch, it will open normally every time!

Step 3: Start Using

Once launched, BetterCountdown will appear in your menu bar at the top of your screen showing "30:00". Click on it to start using your countdown timer!
